正负双电源设计方案

@TOC

正负双电源在电子电路中具有重要作用，特别是在运算放大器、模拟信号处理和一些传感器电路中。它们提供了对称的正电压和负电压，相对于一个共同的接地（GND）。下面是正负双电源的作用以及生成方案的详细说明。

💡正负双电源的作用

  1. 对称电压供电

    • 许多模拟电路，如运算放大器和模数转换器，需要对称的正负电源供电，以便其输出可以在正负范围内摆动。
    • 对称供电可以使信号在零电压附近对称地摆动，提高信号处理的线性度和精度。

  2. 零基准点（GND）

    • 提供一个共同的零电位基准，使电路设计和信号处理更加方便。
    • 在信号处理中，零基准点有助于简化偏移电压的处理和计算。

  3. 提高性能

    • 双电源配置可以提高放大器的动态范围和共模抑制比（CMRR），改善电路性能。
    • 对于一些传感器和精密测量设备，双电源供电有助于提高测量精度和稳定性。

💡生成正负双电源的方案

1、线性电源（变压器+整流+稳压）：

1. 线性电源（变压器+整流+稳压）

  • 变压器：使用中心抽头变压器，其次级绕组提供对称的正负交流电压。
  • 整流电路：将交流电压整流为直流电压。使用全波整流或桥式整流。
  • 滤波电容：平滑整流后的脉动直流电压。
  • 稳压器：使用线性稳压器（如78xx和79xx系列）来提供稳定的正负直流电压。

优点：输出电压稳定、噪声低。 缺点：效率较低、体积较大、散热要求高。

2. 开关电源（DC-DC转换器）：

  • 升压/降压转换器：使用升压或降压转换器将单电源转换为双电源。
  • 反激式转换器：使用反激式转换器生成正负电压。
  • 电荷泵转换器：使用电荷泵技术在低功率应用中生成正负电压。

优点：效率高、体积小、适合便携设备。 缺点：设计复杂、输出噪声较高（需要额外的滤波）。

3. 双极性电源模块：

  • 使用集成的双极性电源模块，可以从单一电源输入生成正负双电源。这些模块通常集成了 开关转换器和稳压电路，提供简单易用的解决方案。

优点：设计简单、使用方便。 缺点：成本较高、选择范围有限。

4. 电池供电：

  • 使用两节串联的电池（如两节9V电池），中间连接点作为接地（GND），两端提供正负电压。
  • 可用于便携式设备或需要隔离供电的应用。

优点：便携、易于实现。 缺点：电池寿命有限、需要定期更换或充电。

👽本人常用设计方案

要求：我对双电源的应用场景是用于传感器采集电路中的运放正负电源供电，需要着重考虑噪声低，纹波尽可能小，满足供电电流的需求。以下是经过使用验证的电路设计和总结。

正电源设计方案

1. 低压差,小电流常用LDO：

LDO的工作原理

正电压的低压差线性稳压器（LDO）是一种电源管理器件，用于从较高的输入电压稳定输出较低的正向电压。它通过调节其控制电路中的传输电阻来维持一个稳定的电压差，适用于对输出电压精度要求较高的应用。

LDO的优点

  • 低噪声：适合对噪声敏感的应用。
  • 简单稳定：没有开关频率引起的电磁干扰。
  • 快速响应：适合需要快速动态响应的应用。
  • 低成本：相对于开关稳压器，成本较低。

LDO的推荐芯片

以下是几款常用的LDO芯片，适合不同的应用需求：

  1. AMS1117

    • 输入电压范围：4.2V 至 15V
    • 输出电压范围：1.2V 至 13.8V
    • 最大输出电流：1A
    • 特点：价格低廉，适用于一般电源管理需求。

  2. MIC5205

    • 输入电压范围：2V 至 16V
    • 输出电压范围：0.8V 至 5V
    • 最大输出电流：150mA
    • 特点：低压差，低噪声，适合噪声敏感的应用。

  3. LT1763

    • 输入电压范围：2.2V 至 20V
    • 输出电压范围：0.8V 至 15V
    • 最大输出电流：500mA
    • 特点：高精度，低噪声，快速动态响应，适合需要高稳定性和低噪声的应用。

  4. LD1117

    • 输入电压范围：4.2V 至 20V
    • 输出电压范围：1.8V 至 15V
    • 最大输出电流：800mA
    • 特点：稳定性高，价格实惠，广泛用于各种电源管理应用。

这些芯片具有不同的输入电压范围、输出电压范围和最大输出电流能力，可以根据具体的应用需求选择合适的LDO芯片。在选择时，请注意考虑电源电压范围、输出电流需求、稳定性和噪声要求等因素，并查阅相关的数据手册以获取更详细的信息。

1. 大压差,大电流常用DC-DC：

DC-DC降压芯片工作原理

DC-DC降压转换器通过以下步骤将输入电压转换为较低的输出电压：

  1. 开关调制: 输入电压被交替地接通和断开，通过开关元件（通常是MOSFET）控制。这种周期性的开关操作在转换过程中产生脉冲。

  2. 变压器原理: 通过变压器或电感器来存储能量，变换电压。当开关关闭时，电能由电感器传递到负载电容器，以稳定输出电压。

  3. 反馈机制: 使用反馈回路来监测输出电压，并根据需要调整开关频率和占空比，以维持稳定的输出电压。

  4. 控制电路: 包括开关驱动器、反馈网络和误差放大器等部件，用于确保输出电压的稳定性和响应性。

推荐使用：TPS5430 DC-DC降压芯片介绍

TPS5430是TI（德州仪器）推出的一款高效率同步降压转换器，适用于各种应用，包括通信设备、工业控制以及便携式设备。以下是它的主要特性：

  • 输入电压范围: 4.5V 至 28V
  • 输出电压范围: 0.8V 至 24V
  • 最大输出电流: 3A
  • 开关频率: 可调节至2.5MHz
  • 效率: 高达95%
  • 保护功能: 包括过载保护、过热保护和欠压锁定

TPS5430的优点

  • 高效率: 提供高达95%的转换效率，有助于减少能源损耗和热量产生。
  • 宽输入电压范围: 适合多种电源供应情况。
  • 可调频率: 可根据应用需要调整开关频率，以优化性能。
  • 内建保护: 包括多种保护功能，提高系统的可靠性和稳定性。 选择TPS5430时，请根据具体的输入电压范围、输出电流需求和功率效率等因素进行评估，并确保充分理解其数据手册中的电路设计和应用注意事项。

负电源设计方案

1. 基于ICL7660AIBAZA-T负电源：

ICL7660AIBAZA-T 是一款电荷泵型 DC-DC 转换器，可以将输入正电压（+V）转换为相等的负电压（-V）。其主要特点如下：

  • 输入电压范围：+1.5V 至 +10V
  • 输出电压范围：-1.5V 至 -10V
  • 输出电流：最大 35mA
  • 低静态电流：典型值 80µA
  • 内部振荡器频率：典型值 10kHz

ICL7660AIBAZA-T 通过内部电荷泵电路实现电压转换。电荷泵电路利用开关电容技术，将输入电压反相后输出。其工作原理主要包括两个阶段：

  1. 电荷转移阶段：在开关操作下，将电容器中的电荷转移到输出端。
  2. 电荷储存阶段：电荷泵电路的电容器重新充电，准备进行下一次电荷转移。

2. LM2776 使用说明

主要特性和规格

  • 输入电压范围: 2.7V 至 5.5V
  • 输出电压范围: -1.25V 至 -6V（取决于输入电压）
  • 最大输出电流: 500mA
  • 开关频率: 1.5MHz
  • 效率: 高达80%
  • 低静态电流: 仅0.01μA
  • 保护特性: 过载保护和短路保护 优点：体积小，纹波小，易实现。外设少，简单。工作频率高达1.5MHz，有助于减小输出电容器的尺寸和成本，提高稳定性和响应速度。

LM2776作为一款高效、低功耗的负压转换器，适合许多低功率应用场合。然而，设计者应根据具体的应用需求和电源要求，仔细评估其优点和缺点，以确保选择合适的电源解决方案。

SGM3204YN6G/TR 是一款电荷泵型 DC-DC 转换器，可以将输入正电压（+V）转换为相等的负电压（-V）。其主要特点如下：

  • 输入电压范围：+1.8V 至 +5.5V
  • 输出电压范围：-1.8V 至 -5.5V
  • 输出电流：最大 100mA
  • 低静态电流：典型值 100µA
  • 开关频率：典型值 250kHz

SGM3204YN6G/TR 通过内部电荷泵电路实现电压转换。电荷泵电路利用开关电容技术，将输入电压反相后输出。其工作原理主要包括两个阶段：

  1. 电荷转移阶段：在开关操作下，将电容器中的电荷转移到输出端。
  2. 电荷储存阶段：电荷泵电路的电容器重新充电，准备进行下一次电荷转移。

优点：体积小，易实现。 缺点：电流大部分条件下能满足运放供电，噪声一般。不方便购买

4. 基于LTC3261的负电源开发说明

主要特性和功能

  • 输入电压范围：
    • VIN：2.7V 至 18V
  • 输出电压范围：
    • VOUT：-1.5V 至 -12.5V
  • 最大输出电流：
    • 50mA
  • 工作频率：
    • 典型开关频率：100kHz
  • 特性：
    • 集成了开关电容式DC-DC转换器和低压差线性稳压器功能。
    • 内部集成了电荷泵和低压差稳压器，简化了电路设计并减少了外部元件的数量。
  • 适用场景：
    • 适合电池供电系统、传感器接口和其他需要负电源的应用。

优点：体积小，易实现，电流大。 缺点：价格太高。
